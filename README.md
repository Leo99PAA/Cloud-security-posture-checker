# Cloud-security-posture-checker
Este projeto é uma ferramenta de linha de comando (CLI) escrita em Python que realiza uma checagem automatizada de postura de segurança em uma conta AWS. Ele coleta dados via AWS APIs com permissões de leitura (IAM ReadOnly) e aplica um conjunto de regras para identificar riscos comuns de configuração, gerando um relatório com achados, evidências e recomendações de correção. Ele não faz nenhuma alteração dentro do sistema AWS, faz somente a leitura dos dados.
